How event loop works in Javascript

  • Event loop is an endless loop, which waits for tasks, executes them and then sleeps until it receives more tasks.
  • The event loop executes tasks from the event queue only when the call stack is empty
  • A thread, the function at the top of the stack is what's currently running. The event loop is at the bottom of the stack
  • An event loop is often the main loop in a program that typically waits for the user to trigger something

How many ways are there to get undefined

  • When a variable has not been assigned a value then we see undefined
  • A function returns undefined if a value was not returned
  • There is a variable or constant which is not set.
  • A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value
